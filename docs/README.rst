============
Book Tracker
============

Functionality
=============
The Book Tracker app should be a one-stop app for recording, cataloging, and analyzing all book data.

Entry Fields
------------
The GUI should be able to capture:
* Metadata
* Metrics
* Reviews

Cataloging
----------
The app should save the entries in a SQL database:
* Book
    - ID
    - AuthorID
    - PublisherID
    - Name
    - Pages
    - StartDate
    - FinishDate
    - rating
